# dotfiles

My personal dotfiles managed by [deadc0de6/dotdrop](https://github.com/deadc0de6/dotdrop).

Need to install the dependent softwares, including but not limited to:
- nodejs
- npm
- yarn
- curl
- git
- python3
- tmux
- neovim >= v0.3.8
- ranger
- powerline-status
- fishshell
- psutils
- true color fonts
